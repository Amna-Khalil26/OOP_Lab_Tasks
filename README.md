# 📘 OOP Lab Tasks (C++)

This repository contains my Object-Oriented Programming (OOP) lab tasks implemented in C++.

## 📂 Lab Task: Smart Healthcare System

### 📌 Description
This program is a simple healthcare management system that manages patients, doctors, appointments, and billing.

### 🧠 Concepts Used
- Classes and Objects
- Inheritance (Person → Patient, Doctor)
- Encapsulation (getters/setters)
- Polymorphism (virtual functions)
- Operator Overloading (Bill class)
- Dynamic Memory Allocation (new/delete)

### ⚙️ Functionality
- Stores patient and doctor information
- Assigns doctors to patients
- Manages appointments
- Generates and compares bills

### ▶️ How to Run
Compile and run using:
g++ lab-04-healthcare-system.cpp -o app
./app
